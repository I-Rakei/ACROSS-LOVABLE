# Icons Update

During our recent updates, I didn't remove any of the standard icons (like the icons from Lucide or FontAwesome) from your pages. 

The only "icon" that was taken out and replaced was the **default browser dropdown arrow** on the form `<select>` fields.

- **What was taken out:** The native browser dropdown arrow (chevron) for all `<select>` inputs (hidden using CSS `appearance: none`).
- **Name of the icon:** Native Browser Select Arrow
- **What it was replaced with:** A custom, modern **SVG Chevron icon** (`<svg>...<path d='m6 8 4 4 4-4'/>...</svg>`) embedded directly into `styles.css`. This ensures the dropdown arrow looks identical and premium across all browsers (Chrome, Safari, Firefox) rather than changing based on the user's operating system.

If you are referring to a different icon from a previous session, please let me know which one and I'll be happy to provide those details!
